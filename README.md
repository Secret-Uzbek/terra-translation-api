> Implementation-facing API branch for semantic translation, UCOM logic, and
> practical language infrastructure inside the Terra ecosystem.

## Layer role

This repository is an implementation API layer.

It should hold:

- API-facing translation logic;
- practical language infrastructure;
- implementation documentation and release surfaces;
- a clear human entry into what this API actually does.

It should not become:

- a mojibake shell;
- a fake DOI placeholder surface;
- a donor governance repository;
- an undifferentiated archive of unrelated code.
